# homebrew-tap

Homebrew tap for [Dictify](https://github.com/aletisunil/Dictify) and other apps by Sunil Aleti.

## Install Dictify

```sh
brew tap aletisunil/tap
brew install --cask dictify
```

After tapping once, you can also just run:

```sh
brew install dictify
```

### Upgrade

```sh
brew upgrade --cask dictify
```

### Uninstall

```sh
brew uninstall --cask dictify        # remove the app
brew uninstall --zap --cask dictify  # also remove its settings/history
```
